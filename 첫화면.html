프로젝트 소개 : 초보 투자자 & 바쁜 직장인들에게 쉽게 투자할 수 있는 방법론을 제시 하고 시장 수익률을 초과하는 알파를 제공하기 위해 만든 프로젝트이다.

프로젝트 동기 : 최근 10년간 개인과 외국인의 수익률 현황을 비교해 보았을때 개인 투자자들이 수익을 얻지 못한 것을 볼 수 있다.
이것은 개인투자자들이 옳지 못한 방법으로 투자를 하고 있기 때문이다.
그래서 우리는 어떻게 하면 개인 투자자들이 옳은 방법으로 투자하여 수익을 얻을 수 있을까? 라는 것에서 이 프로젝트를 만들게 되었다.



프로젝트 개발환경

  1. R 3.5.3
  2. Atom 1.35.1
  3. Eclipse JavaScript
  4. 운영체제 : 윈도우 64bit
  5. API : FNGuide (공공) API 사용
  6. 설치순서 : Rstudio -> Atom -> Eclipse
